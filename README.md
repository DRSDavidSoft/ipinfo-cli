# ipinfo-cli

> Get current(system) ip information

## installation
```
> npm install ipinfo-cli --global
```

## Usage
```
> ipinfo-cli --help

	Example
		> ipinfo
		┌──────────┬──────────────────────────────────────────┐
		│ Title    │ Value                                    │
		├──────────┼──────────────────────────────────────────┤
		│ ip       │ 8.8.8.8                                  │
		├──────────┼──────────────────────────────────────────┤
		│ hostname │ www.xyz.com                              │
		├──────────┼──────────────────────────────────────────┤
		│ city     │ Mountain View                            │
		├──────────┼──────────────────────────────────────────┤
		│ region   │ California                               │
		├──────────┼──────────────────────────────────────────┤
		│ country  │ US                                       │
		├──────────┼──────────────────────────────────────────┤
		│ loc      │ 37.4192,-122.0574                        │
		├──────────┼──────────────────────────────────────────┤
		│ org      │ XYZ Inc.                                 │
		├──────────┼──────────────────────────────────────────┤
		│ postal   │ 99999                                    │
		└──────────┴──────────────────────────────────────────┘
```
